## How to Compile and Run

### C
gcc filename.c -o Myexe 
for linux : ./Myexe
for Windows : Myexe.exe


### C++

g++ filename.cpp -o output
for linux : ./Myexe
for Windows : Myexe.exe

### Java

javac filename.java ,
